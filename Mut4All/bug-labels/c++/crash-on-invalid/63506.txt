Issue Title:
Clang frontend C++ crash when __builtin_dump_struct is used on pointer to undefined type

Source Code & Bug Trace:
#include <stdio.h>

struct Undefined;

struct Defined {
    Undefined* undef;
};

int main() {
    Undefined* value = nullptr;

}
