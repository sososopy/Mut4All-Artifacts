Issue Title:
Infinite size checker doesn't see through impl Trait, leading to rustc overflow · Issue #38064 · rust-lang/rust · GitHub

Rust Code:
#![feature(conservative_impl_trait)]

trait Quux {}

fn foo() -> impl Quux {
    struct Foo<T>(T);
    impl<T> Quux for Foo<T> {}
    Foo(bar())
}

fn bar() -> impl Quux {
    struct Bar<T>(T);
    impl<T> Quux for Bar<T> {}
    Bar(foo())
}

// effectively:
//     struct Foo(Bar);
//     struct Bar(Foo);
// should produce an error about infinite size

fn main() { foo(); }

Bug Labels:
A-impl-trait, A-type-system, C-bug, I-ICE
