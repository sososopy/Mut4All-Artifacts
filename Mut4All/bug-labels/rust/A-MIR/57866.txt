Issue Title:
ICE with nested match with "Type Alias Enum Variant" feature · Issue #57866 · rust-lang/rust · GitHub

Rust Code:
#![feature(type_alias_enum_variants)]

enum Outer<T> {
    A(T)
}

enum Inner {
    A(i32)
}

type OuterAlias = Outer<Inner>;

fn ice(x: OuterAlias) {
    // Fine
    match x {
        OuterAlias::A(Inner::A(_)) => (),
    }
    // Not fine
    match x {
        OuterAlias::A(Inner::A(y)) => (),
    }
}

fn main() {}

Bug Labels:
A-MIR, A-inference, I-ICE, T-compiler
