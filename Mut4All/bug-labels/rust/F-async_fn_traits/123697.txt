Issue Title:
Internal Compiler Error when building an async closure that calls methods on a referenced variable and a cloned referenced variable in a specific order · Issue #123697 · rust-lang/rust · GitHub

Rust Code:
#![feature(async_closure, async_fn_traits)]

fn main() {}

pub fn test(test: &u64, temp: &u64) {
    let test = test.clone();
    async || {
        temp.abs_diff(12);
        test.clone();
    };
}
