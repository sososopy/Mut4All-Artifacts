Issue Title:
Remove phase separation between specialization graph construction and projection · Issue #32205 · rust-lang/rust · GitHub

Rust Code:
trait Assoc {
    type Output;
}

impl<T> Assoc for T {
    type Output = bool;
}

impl Assoc for u8 {} // <- inherits the non-default type from above

trait Foo {}
impl Foo for <u8 as Assoc>::Output {}  // <- this projection will fail with an ICE
