Issue Title:
ICE with trait alias when alias refers to bare Self · Issue #59029 · rust-lang/rust · GitHub

Rust Code:
trait Svc<Req> {
    type Res;
}

// ICEs:
trait MkSvc<Target, Req> = Svc<Target> where Self::Res: Svc<Req>;

// works:
// trait MkSvc<Target, Req> = Svc<Target> where <Self as Svc<Target>>::Res: Svc<Req>;
